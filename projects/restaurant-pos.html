Restaurant POS

Desktop Application ¬∑ Point of Sale System

A fully offline Restaurant Point of Sale (POS) Desktop Application built with Electron, React, and SQLite. This comprehensive POS system provides complete restaurant management capabilities including order processing, inventory management, reporting, and thermal printer support.

Note: This project was developed with the assistance of Cursor AI, leveraging its powerful code generation and refactoring capabilities to build a production-ready POS system.

‚ö° Key Features

Core Functionality

  • Order Management: Multiple order types (Dine In, Take Away, Delivery, Express) with real-time cart updates, quantity management, and item modification
  • Menu & Inventory Management: Category-based menu organization, complete CRUD operations, stock tracking with low stock alerts, and item availability management
  • Billing System: Automatic subtotal calculation, configurable VAT/Tax rates, grand total calculation, and multiple payment modes tracking
  • Work Period Management: Clock in/out functionality, work period tracking, and session management
  • Reports & Analytics: Daily sales reports with date filtering, tax reports with detailed breakdown, summary statistics, and export capabilities
  • KOT (Kitchen Order Ticket) Display: Real-time order display and kitchen order management

Advanced Features

  • Thermal Printer Support: ESC/POS printer integration, bill receipt printing, KOT printing, and USB printer support
  • User Management: Multi-user support with role-based permissions, user authentication system, and permission-based access control
  • Settings Management: Configurable tax rates, printer settings, currency settings, and application preferences
  • Data Management: Automatic daily database backups, manual backup/restore functionality, data recovery mechanisms, and window state persistence
  • Internationalization: Multi-language support (English/Arabic) with language switching capability
